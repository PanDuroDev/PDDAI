# PDDAI

**Трансформер языковой модели с нуля — 4,69M параметров**

Трансформер-декодер с **4,69M параметров**, созданный с нуля на PyTorch — с GQA, RoPE, SwiGLU, Refresh Gates, KV-кэшированным выводом, мульти-источниковым поиском, инструментами агента, RAG и интеграцией 9Router.

## Возможности

| Категория | Детали |
|---|---|
| **Архитектура** | Группированное внимание запросов (6 голов запросов, 3 KV головы), вращательные позиционные внедрения, SwiGLU FFN, Refresh Gates, KV-кэш |
| **Обучение** | Смешанная точность (AMP), накопление градиентов, оптимизатор Muon, Unlikelihood loss + сглаживание меток, сохранение чекпоинтов |
| **Чат** | Обычный диалог с потоковым выводом токен за токеном и экспортом данных обучения при выходе |
| **Агент** | Рассуждение с инструментами и мульти-источниковый поиск (Wikipedia, Wikidata, arXiv, PubMed, StackExchange, GitHub), калькулятор, чтение файлов |
| **Маршрутизатор поиска** | Интеллектуальная маршрутизация запросов для выбора лучших API-источников и агрегации результатов (ограничение 3000 символов) |
| **Конвейер 9Router** | Внешний AI API сжимает результаты поиска в чистый контекст для локальной модели |
| **Очиститель вывода** | Лёгкая постобработка: удаление управляющих символов, уменьшение повторов, дедупликация предложений, исправление пробелов |
| **RAG** | Генерация с дополнением извлечения — простая и улучшенная версии |
| **AI2AI** | Автономный диалог двух моделей с симуляцией поочерёдных ответов |
| **Экспорт обучения** | Диалоги форматируются через 9Router и сохраняются как высококачественные обучающие данные |

## Быстрый старт

```bash
# Обучение с нуля
python main.py

# Чат с обученной моделью
python chat.py

# Агент с инструментами
python -m agents.agent

# Диалог ИИ-ИИ
python ai2ai.py
```

## Требования

- Python 3.10+
- PyTorch 2.0+
- Рекомендуется GPU с CUDA (работает на CPU)
- Библиотека HuggingFace `datasets`

## Архитектура

- **GQA** — 6 голов запросов, 3 KV головы для эффективной памяти
- **RoPE** — Относительное позиционное кодирование без обучаемых параметров
- **SwiGLU** — FFN с вентилем (Swish × GLU)
- **Refresh Gates** — Послойные селективные обновления состояния
- **KV-кэш** — Генерация O(1) на токен
