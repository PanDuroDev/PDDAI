# PDDAI

**शुरू से निर्मित ट्रांसफॉर्मर भाषा मॉडल — 4.69M पैरामीटर**

PyTorch के साथ शुरू से निर्मित **4.69M पैरामीटर** का डिकोडर-ओनली ट्रांसफॉर्मर — GQA, RoPE, SwiGLU, Refresh Gates, KV-कैश्ड इन्फ्रेंस, मल्टी-स्रोत खोज, एजेंट टूल्स, RAG और 9Router एकीकरण के साथ।

## विशेषताएँ

| श्रेणी | विवरण |
|---|---|
| **आर्किटेक्चर** | ग्रुप्ड क्वेरी अटेंशन (6 क्वेरी हेड, 3 KV हेड), रोटरी पोज़िशनल एम्बेडिंग, SwiGLU FFN, Refresh Gates, KV कैश |
| **प्रशिक्षण** | मिश्रित परिशुद्धता (AMP), ग्रेडिएंट संचय, Muon ऑप्टिमाइज़र, अनलाइकलीहुड लॉस + लेबल स्मूथिंग, चेकपॉइंट सेविंग |
| **चैट** | टोकन-दर-टोकन स्ट्रीमिंग आउटपुट और बाहर निकलने पर प्रशिक्षण डेटा निर्यात |
| **एजेंट** | टूल-सहायक तर्क और मल्टी-स्रोत खोज (विकिपीडिया, विकिडेटा, arXiv, PubMed, StackExchange, GitHub), कैलकुलेटर, फ़ाइल रीडर |
| **खोज राउटर** | क्वेरी-इंटेंट रूटिंग जो सर्वश्रेष्ठ API स्रोत चुनता है और परिणाम एकत्र करता है (3000 वर्ण सीमा) |
| **9Router पाइपलाइन** | बाहरी AI API खोज परिणामों को स्थानीय मॉडल के लिए साफ़ संदर्भ में संपीड़ित करता है |
| **आउटपुट क्लीनर** | हल्का पोस्ट-प्रोसेसिंग: नियंत्रण वर्ण हटाना, पुनरावृत्ति कम करना, वाक्य डिडुप्लिकेशन, स्पेसिंग सुधार |
| **RAG** | रिट्रीवल-ऑग्मेंटेड जनरेशन — सरल और बेहतर संस्करण |
| **AI2AI** | दो-मॉडल स्वायत्त वार्तालाप, टर्न-टेकिंग सिमुलेशन के साथ |
| **प्रशिक्षण निर्यात** | वार्तालाप 9Router के माध्यम से फ़ॉर्मेट होते हैं और उच्च-गुणवत्ता वाले प्रशिक्षण डेटा के रूप में सहेजे जाते हैं |

## त्वरित आरंभ

```bash
# शुरू से प्रशिक्षण
python main.py

# प्रशिक्षित मॉडल के साथ चैट
python chat.py

# टूल्स के साथ एजेंट
python -m agents.agent

# AI-से-AI वार्तालाप
python ai2ai.py
```

## आवश्यकताएँ

- Python 3.10+
- PyTorch 2.0+
- CUDA-सक्षम GPU अनुशंसित (CPU पर चलता है)
- HuggingFace `datasets` लाइब्रेरी

## आर्किटेक्चर

- **GQA** — 6 क्वेरी हेड, 3 KV हेड कुशल मेमोरी के लिए
- **RoPE** — बिना सीखे पैरामीटर के सापेक्ष स्थिति एन्कोडिंग
- **SwiGLU** — गेटेड FFN (Swish × GLU)
- **Refresh Gates** — परत-दर-परत चयनात्मक स्थिति अपडेट
- **KV कैश** — प्रति टोकन O(1) जनरेशन
